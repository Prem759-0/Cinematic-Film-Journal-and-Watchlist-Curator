# Cinematic Film Journal

A beautifully designed, pure-frontend application for logging films, managing a watchlist, and viewing cinematic habits. 

## Features
- Deep amber/gold analog film aesthetic with grain overlay.
- Full client-side persistence (localStorage).
- Responsive 16:9 cinematic card layouts.
- Advanced statistical genre charting.
- Framer Motion integrated transitions.

## Quick Start
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

Built for Vercel deployment.
